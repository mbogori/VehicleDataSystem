# Vehicle Data System

The Vehicle Data System is a Java coursework project that reads and processes vehicle information stored in a CSV file.

The project is being developed step by step to demonstrate important Java programming concepts, including file handling, data processing, methods, classes, objects, collections and error handling.

## Current functionality

- Reads vehicle records from `vehicles.csv`
- Processes the data using Java
- Displays vehicle information in a readable format
- Separates program logic into appropriate methods and classes
- Provides a foundation for adding searching, sorting and validation features

## Project files

- `readVehiclesData.java` — contains the Java code used to read and process the vehicle data
- `vehicles.csv` — contains the vehicle records used by the program

Additional Java classes may be added as the project develops.

## Technologies used

- Java
- CSV files
- Visual Studio Code
- Git and GitHub

## Running the project

1. Clone or download the repository.
2. Open the project folder in Visual Studio Code.
3. Ensure that Java is installed.
4. Keep `vehicles.csv` in the location expected by the Java program.
5. Run the Java class containing the `main` method.

For example:

```bash
javac readVehiclesData.java
java readVehiclesData
